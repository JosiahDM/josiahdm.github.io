Java Basics: void methods and return types

Methods are used to do some action, manipulate some value, and/or do some kind of calculation. Methods help us out because within a method we can write all the instructions we want to happen, and simply write the method wherever we need it. Instead of copy-pasting multiple lines of code (which would be hard to edit later if we had lots of code), we can create a method containing those lines of code, then "call" that method with one line of code, wherever we need to use it.

When we declare a method, we have to declare whether the method is void, or returns some kind of type. So what does that mean?

When a method is "void" it will simply run the instructions within the method when called. A void method is kind of like an "action" method. You simply need something done now.

Here's an example of a void method:

public static void printStars() {
    System.out.println("**********");
}

This method (although not very exciting) will print 10 asterisks when we call it. To call it, we would simply write printStars(); on it's own line.

Another example is the main method in every Java program. The main method is void--it just runs whatever code is inside it.

Return Types

When a method is declared with a return type, it could look like this: 

public static int add(int a, int b) {
    return a + b;
}

This method will take two parameters, both integers, then "return" an integer result of adding them together. Note that the return type must be the same as what the method is declared as (int in this example). An int method can't return a String value, for example.

You can assign values to variables using return methods. So you could do int x = add(4, 5); and x would then store the value that is returned from the add method. In thise case, 9.

You can also use the returning method within other method calls to get the value you want. So maybe you wanted to do something like System.out.println( "Your answer is: " + add(4, 5) ); this would print "Your answer is: 9" because the add method will return 9, and then that will be added to the println() call.

Return methods let us run some kind of calculation, get some value back, and use the value in whatever way we need.

To sum up:

- void methods do not return a value. They just run the code within them.

- methods with a return type run the code, and then return some value. The value returned must be of the type that is declared.

- properly created methods can make code easier to write, read, and maintain.
